# BongBong and Avatar - Dual AI Companion System

An AI-powered dual-bot system designed for elderly care and family communication.

## Features

- Dual Bot Architecture (Professional + Humorous)
- Eye of Truth multi-model verification
- Voice/Image processing
- Smart Memory (MongoDB + Notion)
- Visa Consultation (Grok-powered)

## Tech Stack

- Node.js 20+
- Gemini 2.5 + Grok 3
- MongoDB Vector Search
- Notion API sync
- PM2 + Doppler
